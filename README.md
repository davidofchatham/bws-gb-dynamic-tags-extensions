# BWS GenerateBlocks Dynamic Tags Extensions

WordPress plugin that extends [GenerateBlocks](https://generateblocks.com/) with additional dynamic tags for custom field data, taxonomy terms, and date/time formatting.

Tags are organized by **source** (current post, related post, taxonomy term, etc.) and **template** (text, image, date, etc.), enabling flexible content injection without hardcoding field logic in templates.

## Requirements

- WordPress
- GenerateBlocks (Pro recommended)
- A custom fields plugin (ACF or compatible)
